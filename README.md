# IRIDA Metadata Client

## Usage

```
usage: irida_metadata_client.py [-h] [--base-url BASE_URL] [--client-id CLIENT_ID] [--client-secret CLIENT_SECRET] [--username USERNAME] [--password PASSWORD] [--project-id PROJECT_ID]

optional arguments:
  -h, --help            show this help message and exit
  --base-url BASE_URL
  --client-id CLIENT_ID
  --client-secret CLIENT_SECRET
  --username USERNAME
  --password PASSWORD
  --project-id PROJECT_ID
```
